# High Strangeness Website

Marketing and landing page for the High Strangeness app.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This site is configured to deploy automatically to Vercel when pushed to GitHub.

## Domain

Domain: highstrangeness.app
Hosting: Vercel
DNS: Cloudflare
